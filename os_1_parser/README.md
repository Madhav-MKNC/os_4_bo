# IAC

Software to clean Indian address, pin and phone number

## Install dependencies

```
pip install -r requirements.txt;
```

## Run

```
python main.py -f path/to/address/file.txt
```

Example: <code>python main.py -f WhatsApp_Chat_with_Madhav_28_Nov.txt</code>

## Available commands 

```sh 
Python main.py -f file.txt
python main.py --f file.txt 
Python main.py -file file.txt 
Python main.py --file file.txt 


Python main.py -m file.txt 
Python main.py --m file.txt
Python main.py -modify file.txt 
Python main.py -modified file.txt 


Python main.py -n file.txt 
Python main.py --n file.txt  
Python main.py -name file.txt 
Python main.py --name file.txt 


Python main.py -t file.txt 
Python main.py --t file.txt 
Python main.py -translate file.txt 
Python main.py --translate file.txt
```
